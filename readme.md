Selenium Java Basic
-----------


## Setting Up
- JDK 11
- Maven 3.11
- Chrome
- Firefox

## How to run
```bash
mvn test -Dtest.suite=todomvc-testplan.xml
```
- check the html report at [surefire-report](target/surefire-reports/index.html)
## How to add more test
- suffix `**Test.java`
